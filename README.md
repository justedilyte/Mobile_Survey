# Mobile_Survey

## A Simple Design for a Small Mobile Survey

### Parts of the Survey:

+ Welcome Page

+ Question 1: Single-Choice Question 

+ Question 2: Multi-Choice Question with Choices Containing Images 

+ Question 3: Text-Based Question with One Main Image

+ Question 4: Matrix Question

+ Thank You Page


### View of the Survey

In order to view the survey, open the `index.html` file in your browser and if it has been opened from your local directory in which you have cloned full repository, you should be able to move between the pages via main `Next` and `Back` buttons.
